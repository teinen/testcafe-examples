# Testcafe examples practice

Getting started: https://devexpress.github.io/testcafe/documentation/getting-started

## Install

```bash
$ cd testcafe-examples
$ npm install
```

## Run test

```bash
# chrome
$ npm run test:ch

# safari
$ npm run test:sa

# firefox
$ npm run test:ff

# edge
$ npm run test:ed
```
